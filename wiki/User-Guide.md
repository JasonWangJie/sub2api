# 用户使用指南

[← Wiki 首页](Home.md)

面向拿到 Sub2API 站点账号的终端用户。

---

## 1. 注册与登录

- 邮箱密码（若管理员开启注册）
- 邀请码 / 优惠码（若要求）
- 第三方：GitHub、Google、微信、钉钉、OIDC、LinuxDo 等（以站点设置为准）
- 可开启 TOTP 双因素

登录后进入用户仪表盘，查看余额、用量与 Key。

---

## 2. API Key

1. 打开 **API Keys** 页面  
2. 创建 Key，选择可用 **分组**  
3. 复制 `sk-...`（通常只显示一次）  
4. 可按需设置名称、IP 限制等（视版本功能而定）

**注意：** 分组决定你能用的模型与上游平台。选错组会导致接口 404 或不支持某 API。

---

## 3. 调用方式

详见 [网关 API](Gateway-API.md)。摘要：

| 工具 | 典型配置 |
|------|----------|
| Claude Code | `ANTHROPIC_BASE_URL` + `ANTHROPIC_AUTH_TOKEN` |
| OpenAI SDK | `base_url=https://host/v1` + `api_key` |
| Gemini CLI | 指向 `/v1beta` |
| Codex | `/v1/responses` 或 `/backend-api/codex` |

查询 Key 侧计费：`GET /v1/sub2api/billing`（带 Key）。

---

## 4. 余额与订阅

- **按量余额**：每次调用按 Token × 倍率扣费  
- **订阅套餐**：若站点提供，可在订阅页购买/查看额度  
- **兑换码**：在兑换页输入 Redeem Code 充余额或时长  
- **在线充值**：支付页选择支付宝 / 微信 / Stripe 等（管理员开启后可见）

用量页可按时间筛选调用记录与花费。

---

## 5. 其它用户功能

| 功能 | 说明 |
|------|------|
| 个人资料 | 改密码、绑定 OAuth、安全设置 |
| 推广/邀请 | 若开启 Affiliate |
| 渠道状态 | 查看公开的渠道可用性（若开启） |
| 批量图片 | 批量提交图片任务（需站点支持与存储配置） |
| 订单 | 查看充值订单状态 |

---

## 6. 常见报错（用户侧）

| 现象 | 可能原因 |
|------|----------|
| 401 / invalid key | Key 错误、已删、未生效 |
| 需绑定分组 | Key 未选 Group |
| 余额不足 | 充值或找管理员调额 |
| 429 / 并发限制 | 触发用户或账号 RPM/并发 |
| 模型不存在 | 分组未放行该模型 |
| 异步图片 404 | 站点未开对象存储 |

更多见 [FAQ](FAQ.md)、[运维与排障](Operations.md)。
